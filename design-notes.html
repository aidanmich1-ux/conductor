Design notes & open decisions

Everything on this page comes from the annotations on the original concept PDF. Nothing here is finished copy, it's a working reference for decisions still to confirm.

Colour palette

Exact values from the Brand Guidelines (p.19), Pantone 7604 C / 1815 C / Black C.

Red #821619
Black #000000
Cream #EDEADF
Annotation accent (not in brand guidelines, my own addition for tracking concept notes) #E619AE

Typography

Per the Brand Guidelines type hierarchy (p.21).

  • Heading / caption, all caps: TT Interphases Pro Mono is specified but has not been supplied. Space Mono is standing in until the licensed font file arrives, everything using it is flagged so it's a one-line swap later.
  • Body copy: EB Garamond, loaded from Google Fonts (free, open licence, matches "EB Garamond - Full Suite" in the guidelines exactly).
  • Hero / annotations / script moments: Blissful Radiance, the supplied webfont, now used for "let's be friends!", "nice to meet you!", the homepage tagline, and other handwritten brand-voice asides.
  • Title cards (e.g. the Brotherwolf project overlay): Bookmania SemiBold for the title, Neue Haas Grotesk Display 65 Medium for the small caps sub-line, both supplied and licensed (see the Commercial Type receipt for the Neue Haas licence, web + desktop, up to 15K monthly uniques).

Footer colour option

The concept shows three footer colourways. Red is live sitewide, use this to compare against black and cream.

Open decisions from the annotations

  • Hero: real brand video now wired in (autoplay, muted, looped, no controls). Possible logo animation on load is still an open idea, not built.
  • Statement section: napkin video with text, or the simpler option of text on a plain background (built as the live default here, still a placeholder-free plain background until a napkin video edit exists).
  • Subscription / Campaign split: each half should play video on hover on desktop. Built as a hover zoom for now, swap in real footage when it exists.
  • Mobile gallery should run two columns rather than three, already handled in the responsive grid.
  • Contact email appears two ways in the source file: hello@penelopestudio.com.au (footer, used sitewide here) vs hello@penelopestudios.com.au (contact page mockup, with an "s"). Worth confirming the correct one before launch.
